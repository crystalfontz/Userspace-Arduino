# ADC Specification

**Default resolution of ADC is 10.**


There are seven ADCs (LRADC0 to LRADC6) available on CFA10037 board.

| ADC Pin | Arduino Uno Pin no | i.MX28 Ball no | CFA10037 Pin no |
| :--: | :--: | :--: |  :--: |
| 0 |  A0  |  C15  | LRADC0 |
| 1 |  A1  |  C09  | LRADC1 |
| 2 |  A2  |  C08  | LRADC2 |
| 3 |  A3  |  D09  | LRADC3 |
| 4 |  A4  |  D13  | LRADC4 |
| 5 |  A5  |  D15  | LRADC5 |
| 6 |  A6  |  C14  | LRADC6 |

```NOTE: Input range for ADC is 0 to 1.85V```
