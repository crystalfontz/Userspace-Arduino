# SPI Specifications

There is one ``spidev`` compatible SPI channel in CFA10036 with three Slave Select (SS) lines.

### Pinout Table

| SPI Pin | Arduino Uno Pin no | Ball no of i.MX28 | Pin no in CFA10037 |
| :--: | :--: | :--: |  :--: |
| MISO |  12  |      |       |
| MOSI |  11  |  L9  | P0.28 |
| SCK  |  13  |  R6  | P0.24 |
| SS0  |  11  |  N9  | P0.17 |
| SS1  |  --  |  P6  | P0.26 |
| SS2  |  --  |  P7  | P0.27 |
